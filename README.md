# 📋 Design a Parent-Teacher Conference Form

A responsive form project built as part of the **freeCodeCamp Responsive Web Design Workshop**. This project focuses on creating a well-structured and user-friendly parent-teacher conference registration form using HTML and CSS.

## 📌 Project Overview

The **Design a Parent-Teacher Conference Form** workshop demonstrates how to create and style forms for collecting information efficiently. It emphasizes accessibility, layout, and responsive design principles.

## 🚀 Features

* Responsive form layout
* Input fields for parent and student information
* Dropdown menus and selection options
* Radio buttons and checkboxes
* Submit button with clean styling
* Organized and readable code

## 🛠️ Technologies Used

* HTML5
* CSS3
